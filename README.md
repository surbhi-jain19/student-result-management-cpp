# Student Result Management System (C++)

A console based application to manage results of multiple students.

## Features
- Input student details
- Calculate total and percentage
- Assign grades
- Display student results

## Concepts Used
- Structures
- Functions
- Arrays
- Loops
- Conditional Statements

## Platform
- Language: C++
- IDE: Code::Blocks

## Class-Based Version (OOP)
- Refactored the program using classes and objects
- Implemented encapsulation using private data members
- Used member functions for input, calculation, grading, and display
